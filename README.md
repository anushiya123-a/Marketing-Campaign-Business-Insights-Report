# Marketing Campaign Business Insights Report (EDA)

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a marketing campaign dataset using Microsoft Power BI.

## Tools Used
- Microsoft Power BI
- Microsoft Excel

## Dataset
Marketing Campaign Dataset

## Dashboard Features
- Total Campaigns
- Total Clicks
- Total Impressions
- Average ROI
- Average Conversion Rate
- ROI by Marketing Channel
- Clicks by Marketing Channel
- Campaign Type Distribution
- Customer Segment Analysis
- Location Analysis
- Scatter Plot
- Business Insights
- Recommendations

## Project Outcome
The dashboard helps identify high-performing marketing channels and provides recommendations for budget allocation based on ROI.
